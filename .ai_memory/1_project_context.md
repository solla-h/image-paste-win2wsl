# 项目核心知识库

## 项目目标
解决 Windows 用户在 WSL2 环境下使用 CLI Agent 时，无法直接将 Windows 剪贴板中的图片传递给 AI 工具的问题。

## 核心共识
1. **工作流**：用户按下 `Alt+V` -> 工具后台保存剪贴板图片 -> 自动输入 WSL 兼容路径 (`/mnt/c/...`) 到当前终端。
2. **技术栈**：AutoHotkey (v2) 用于热键和粘贴逻辑，PowerShell 用于图片保存和文件操作。
3. **版本特性 (v2.0)**：
    - **极速响应**：路径优先粘贴，图片异步保存。
    - **输入法保护**：防止中文输入法干扰路径粘贴。
    - **自动清理**：定时清理临时文件。
